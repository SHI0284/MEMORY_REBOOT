---

## 🎮 Overview

Memory Reboot는  
현실과 가상이 뒤섞인 세계 속에서 플레이어가 미션을 수행하며  
점차 진실에 다가가는 **스토리 중심 액션 게임**입니다.

플레이어는 가상세계와 현실세계를 오가며 문제를 해결하고,  
점점 두 세계의 경계가 무너지는 경험을 하게 됩니다.



---

## 🧠 Core Concept

- 🧩 현실 vs 가상 세계 구조  
- 🔄 세계 전환 기반 진행  
- 📖 스토리 중심 설계  
- 🧠 인식 반전 구조  

👉 현실이라고 믿었던 것이 사실은 가상이었다는 반전 구조

## ⚙️ Key Systems

### 👁 Detection System

- BoxCast 기반 플레이어 감지  
- detectionLevel 값 증가에 따른 UI 변화  
- Animator와 연동된 시각적 피드백  

---

### ⚡ Dash Attack System

- 방향 벡터 기반 대쉬 이동  
- dashCooldown / dashDuration으로 상태 제어  
- 범위 내 적 탐지 후 데미지 적용  


### 💥 Area Attack System
- 일정 반경(radius) 내 적 탐색
- Enemy 태그 기반 필터링
- Animator 기반 공격 연출

### 🎮 UI System
- HP / 상태 UI
- 발각도 시스템 UI
- 애니메이션 기반 피드백
